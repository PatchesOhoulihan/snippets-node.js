Optionen: npm install -g modul für globale installation

Sucht nach einem Nodepacket. Hierfür wird eine Liste von npmjs.com herunter geladen. Das Format dieser Indexdatei ist JSON. Die erste Suchanfrage die nach der Installation erfolgt länger da die erwähnte Datei erst angefordert werden muss.
npm install modul

Installiert das betreffende Modul. Bei globaler Installation werden die Abhängigkeiten in usr/local/lib/node_modules installiert. Grund für eine globale installation sollte vorrangig die CLi sein. Packet suche bei lokal installierten schneller + Abhängigkeit nach außen, sprich wenn die Applikation auf einem anderen Rechner ausgeführt wird müssen die Abhängigkeiten dort auch installiert sein.
npm list
Optionen: npm list -g
npm list --depth=n

Listet die lokal installierten Abhängigkeiten auf. Mit dem Schalter -g werden auch die Global installierten packete aufgelistet. --depth Gibt die Tiefe in der die Packet abhängigkeiten angegeben werden.
